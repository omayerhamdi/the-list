# দ্যা লিস্ট (The List)

> পরিবারের বাজার-সদাইয়ের তালিকা এক জায়গায় বেছে নিন, গোছানো তালিকাটা কপি করে বা হোয়াটসঅ্যাপে পাঠিয়ে দিন — টাইপ করার ঝামেলা ছাড়াই।

**লাইভ:** https://the-list.omayerhamdi.workers.dev/
**ধরন:** স্ট্যাটিক সিঙ্গেল-পেজ ওয়েব অ্যাপ (একটিমাত্র `index.html`)
**ভাষা:** সম্পূর্ণ বাংলা — UI, আউটপুট, সংখ্যা সবকিছু

---

## ১. এই অ্যাপটা আসলে কী

"দ্যা লিস্ট" একটা ছোট, দ্রুত ওয়েব-টুল যা দিয়ে বাসার বাজারের তালিকা বানানো যায়। কোনো
অ্যাকাউন্ট লাগে না, লগইন লাগে না, অ্যাপ ইনস্টল করা লাগে না — শুধু লিংকে ঢুকে জিনিস বেছে
নিলেই তালিকা তৈরি। এরপর এক ট্যাপে পুরো তালিকা কপি করা যায়, অথবা হোয়াটসঅ্যাপে শেয়ার
করা যায় (যাকে ইচ্ছা তাকে)।

পুরো অ্যাপটা একটামাত্র HTML ফাইল — কোনো ফ্রেমওয়ার্ক নেই, বিল্ড স্টেপ নেই, ডেটাবেজ নেই,
সার্ভার নেই। খুবই হালকা, মোবাইলে সাথে সাথে খোলে।

---

## ২. কেন বানানো হয়েছে (যে সমস্যার সমাধান)

পরিবারের যিনি বাজারের দায়িত্বে থাকেন (যেমন আম্মা, ভাবি, স্ত্রী), তিনি প্রতি মাসে/সপ্তাহে
কী কী লাগবে তা হোয়াটসঅ্যাপে টাইপ করে কাউকে (স্বামী, ভাই, বা যিনি বাজার করবেন) পাঠান।

সেই টাইপ করার প্রক্রিয়াটা ছিল ঝামেলার:
- একটা একটা করে টাইপ করতে হতো — সময়সাপেক্ষ
- বানান ভুল হতো, পড়তে কষ্ট হতো
- এলোমেলো, কোনো গোছানো ফরম্যাট থাকত না

**দ্যা লিস্ট এই সমস্যাটাই সমাধান করে:** টাইপ না করে ট্যাপ করে বেছে নিন → পরিমাণ ঠিক করুন →
গোছানো ও সুন্দর ফরম্যাটে তালিকাটা কপি/শেয়ার করে দিন।

## ৩. কার জন্য

মূল ব্যবহারকারী বাংলাদেশি পরিবারের সদস্য — বেশিরভাগ ক্ষেত্রে নারী সদস্য, মোবাইল থেকে
ব্যবহার করবেন। তাই ডিজাইনটা **mobile-first**, বড় ট্যাপ-টার্গেট, সহজ বাংলা, এবং কম ধাপের।

## ৪. কীভাবে কাজ করে (ইউজার ফ্লো)

```
অ্যাপ খোলা (লিংকে ঢোকা)
      ↓
"নিয়মিত" সেকশন বা কোনো ক্যাটাগরি ব্রাউজ করা / সার্চ করা
      ↓
আইটেমে এক ট্যাপ → সিলেক্ট হয় (ডিফল্ট পরিমাণসহ)
      ↓

      ↓

      ↓

      ↓

    অথবা
"হোয়াটসঅ্যাপে পাঠান" — হোয়াটসঅ্যাপ খোলে, তালিকা আগেই লেখা থাকে, যাকে ইচ্ছা তাকে পাঠান
```

## ৫. ফিচার তালিকা

- **ক্যাটাগরি-ভিত্তিক ক্যাটালগ** — ১১টি ক্যাটাগরি, ১২৫+ সাধারণ আইটেম
- **"নিয়মিত" সেকশন** — প্রতিবার যা লাগে, সবচেয়ে সাধারণ জিনিসগুলো এক জায়গায়
- **এক-ট্যাপ অ্যাড** — ট্যাপ করলেই ডিফল্ট পরিমাণসহ যোগ হয়
- **স্টেপার (+/−)** — কিবোর্ড ছাড়াই পরিমাণ বাড়ানো-কমানো
- **সার্চ** — বাংলায় নাম লিখে ফিল্টার
- **নিজে যোগ করা** — তালিকায় নেই এমন জিনিস প্রতিটি সেকশনের নিচে লিখে যোগ করা যায়;
  ক্যাটাগরিতে যোগ করলে সেই ক্যাটাগরির নিচে, "নিয়মিত"-এ যোগ করলে "অন্যান্য" গ্রুপে বসে
- **ভাসমান কর্নার কার্ট** — কী কী বেছেছেন তা সবসময় নিচে-ডানে একটি ব্যাজে স্পষ্ট দেখায়,
  নতুন যোগ হলে হালকা অ্যানিমেশন দেয় (শপিং সাইটের মতো)
- **রিভিউ শিট** — পাঠানোর আগে পুরো তালিকা ক্যাটাগরি অনুযায়ী দেখা, পরিমাণ বদলানো, বাদ দেওয়া
- **কপি বাটন** — পুরো তালিকা ক্লিপবোর্ডে
- **হোয়াটসঅ্যাপ শেয়ার** — নির্দিষ্ট কোনো নম্বরে নয়; যিনি পাঠাচ্ছেন তিনি নিজের হোয়াটসঅ্যাপ
  থেকে যাকে ইচ্ছা তাকে পাঠাতে পারেন
- **বাংলা সংখ্যা** — সব সংখ্যা বাংলায় (১, ২, ৩…)
- **সুন্দর মেসেজ ফরম্যাট** — বোল্ড ক্যাটাগরি হেডারসহ গোছানো তালিকা
- **Mobile-first, responsive** — সব ডিভাইসে ভালো দেখায়
- **সোশ্যাল প্রিভিউ (Open Graph)** — লিংক শেয়ার করলে সুন্দর কার্ড-প্রিভিউ দেখায়

## ৮. ডেটা মডেল (ক্যাটালগ কীভাবে সাজানো)

`index.html`-এর `<script>`-এ দুটি জিনিস:

**ক্যাটাগরি:**
```javascript
{ id:"cat_04", name:"কাঁচাবাজার ও সবজি", short:"সবজি", eb:"তাজা খাবার", ic:"🥬" }
// id = অভ্যন্তরীণ আইডি, name = পুরো নাম, short = ট্যাবের ছোট নাম,
// eb = eyebrow (ছোট লেবেল), ic = ইমোজি আইকন
```

**আইটেম** (প্রতি ক্যাটাগরির নিচে অ্যারে হিসেবে):
```javascript
["নাম", "একক", ডিফল্ট_পরিমাণ, ধাপ, সর্বনিম্ন, নিয়মিত?]
```
উদাহরণ:
```javascript
["মিনিকেট চাল", "কেজি", 25, 5, 1, 1]   // নিয়মিত আইটেম, ডিফল্ট ২৫ কেজি, ধাপ ৫, সর্বনিম্ন ১
["করলা", "কেজি", 0.5, 0.5, 0.5]        // অনিয়মিত, ডিফল্ট ০.৫ কেজি
```
> শেষের `1` থাকলে আইটেমটি "নিয়মিত বাজার" সেকশনেও দেখায়।

## ৯. কীভাবে পরিবর্তন করবেন

সব পরিবর্তন `index.html`-এর ভেতরেই।

- **নতুন আইটেম যোগ:** সংশ্লিষ্ট ক্যাটাগরির `CAT_ITEMS`-এ নতুন লাইন যোগ করুন
  (ফরম্যাট উপরে সেকশন ৮-এ)।
- **নতুন ক্যাটাগরি যোগ:** `CATEGORIES`-এ নতুন অবজেক্ট ও `CAT_ITEMS`-এ ওই আইডির আইটেম-অ্যারে।
- **রঙ বদলানো:** `<style>`-এর `:root` অংশে CSS ভেরিয়েবল (`--clay`, `--bg` ইত্যাদি)।
- **টাইটেল/ট্যাগলাইন/ডেসক্রিপশন:** `<head>`-এর মেটা ট্যাগ ও হেডারের `.tagline`।
- **হোয়াটসঅ্যাপ শেয়ার:** এখন কোনো নির্দিষ্ট নম্বর নেই — `sendWhatsApp()` ফাংশনে
  `https://wa.me/?text=...` ব্যবহার করে, যাতে ইউজার নিজেই প্রাপক বেছে নেন।

## ১০. ডিপ্লয়

- **হোস্টিং:** Cloudflare Workers, GitHub রেপো `omayerhamdi/the-list`-এর সাথে যুক্ত।
- **প্রক্রিয়া:** `main` ব্রাঞ্চে পুশ করলেই Cloudflare অটো-ডিপ্লয় করে। আলাদা বিল্ড কমান্ড নেই।
- **লাইভ URL:** https://the-list.omayerhamdi.workers.dev/

> **গুরুত্বপূর্ণ:** Open Graph ট্যাগে (`og:image`, `og:url`) যে ডোমেইন লেখা আছে, সেটা অবশ্যই
> আসল লাইভ ডোমেইনের সাথে মিলতে হবে — নাহলে হোয়াটসঅ্যাপ/ফেসবুকে শেয়ার-প্রিভিউতে ছবি আসবে না।
> ডোমেইন বদলালে `index.html`-এর `<head>`-এ ওই URL-গুলো আপডেট করতে হবে।

### Open Graph ইমেজ কীভাবে বানানো/বদলানো হয়

`og-image.png` হাতে আঁকা নয় — এটি `og.html` থেকে রেন্ডার করা:

1. `og.html`-এ লেখা/ডিজাইন এডিট করুন (১২০০×৬৩০ সাইজে ডিজাইন করা)।
2. হেডলেস Chromium দিয়ে ঠিক ১২০০×৬৩০ সাইজে স্ক্রিনশট নিয়ে `og-image.png` বানান:
   ```bash
   chromium --headless --window-size=1200,630 \
     --screenshot=og-image.png og.html
   ```
3. `og-image.png` কমিট করে পুশ করুন।
4. স্ক্র্যাপারের ক্যাশ এড়াতে `<head>`-এর `og:image` URL-এ ভার্সন বাড়ান
   (যেমন `?v=3` → `?v=4`)।
5. হোয়াটসঅ্যাপ/ফেসবুকে সাথে সাথে আপডেট দেখতে
   [Facebook Sharing Debugger](https://developers.facebook.com/tools/debug/)-এ লিংক দিয়ে
   "Scrape Again" চাপুন।

> **প্রিভিউ না এলে যাচাই:** ব্রাউজারে সরাসরি `https://the-list.omayerhamdi.workers.dev/og-image.png`
> খুলে দেখুন ছবিটা লোড হয় কিনা। না হলে Worker স্ট্যাটিক ফাইল সার্ভ করছে না — তখন
> স্ট্যাটিক-অ্যাসেট কনফিগ (wrangler) যোগ করতে হবে।

## ১৩. প্রজেক্ট ইতিহাস

- প্রথম তৈরি: Claude.ai ওয়েব চ্যাটে Omayer Hamdi ও Claude-এর সহযোগিতায় (আগের নাম "বাজার")
- ভিত্তি ডেটা: পরিবারের আসল হোয়াটসঅ্যাপ বাজার লিস্ট (আম্মা ও ভাবির কয়েক মাসের মেসেজ),
  পরে বাংলাদেশি পরিবারের সাধারণ প্রয়োজন বিবেচনায় সম্প্রসারিত ও পরিচ্ছন্ন করা
- পরে "দ্যা লিস্ট" নামে রিব্র্যান্ড; ক্যাটালগ পরিষ্কার (ব্র্যান্ড নাম বাদ, ভিন্ন পণ্য আলাদা),
  ভাসমান কার্ট, প্রতি সেকশনে নিজে-যোগ, এবং নম্বরহীন হোয়াটসঅ্যাপ শেয়ার যুক্ত
