# দ্যা লিস্ট — প্রজেক্ট পরিচিতি (Claude Code-এর জন্য)

## প্রজেক্টের সংক্ষিপ্ত পরিচয়

**প্রজেক্টের নাম:** দ্যা লিস্ট (The List) — আগের নাম ছিল "বাজার"
**ধরন:** Static Single-Page Web Application
**ভাষা:** সম্পূর্ণ বাংলা (UI, আউটপুট, সংখ্যা — সবই বাংলায়)
**মূল ফাইল:** `index.html` (একটিমাত্র ফাইল — কোনো framework, build step, বা dependency নেই)
**স্ট্যাক:** Vanilla HTML + CSS + JavaScript
**ডিপ্লয়:** Cloudflare Pages (GitHub push হলেই অটো-ডিপ্লয়)

---

## কেন এই অ্যাপ তৈরি হয়েছে

পরিবারের সদস্যরা (আম্মা, ভাবি, স্ত্রী) প্রতি মাসে বাজারের লিস্ট হোয়াটসঅ্যাপে টাইপ করে পাঠান।
সেই প্রক্রিয়াটা ছিল এলোমেলো, বানান-ভুলে ভরা, এবং পড়তে কঠিন।

**এই অ্যাপ সেই সমস্যার সমাধান করে:**
- তারা অ্যাপ খুলবেন → আইটেম ট্যাপ করে সিলেক্ট করবেন → পরিমাণ ঠিক করবেন → এক বাটনে হোয়াটসঅ্যাপে পাঠাবেন
- টাইপ করার ঝামেলা নেই, বানান-ভুলের ঝামেলা নেই
- আউটপুট সুন্দর ফরম্যাটে WhatsApp-এ পৌঁছায় (বোল্ড হেডারসহ)

**মূল ব্যবহারকারী:** বাংলাদেশি পরিবারের নারী সদস্য, মোবাইল থেকে ব্যবহার করবেন।

## ইউজার ফ্লো (কীভাবে কাজ করে)

```
অ্যাপ খোলা
    ↓
ক্যাটাগরি ব্রাউজ করা অথবা সার্চ করা
    ↓
আইটেমে এক ট্যাপ → সেলেক্ট হয় (ডিফল্ট পরিমাণসহ)
    ↓
+/− বাটনে পরিমাণ বাড়ানো/কমানো
    ↓
"অন্যান্য জিনিস" বক্সে কাস্টম আইটেম লেখা (যা ক্যাটালগে নেই)
    ↓
নিচের স্টিকি বারে "হোয়াটসঅ্যাপে পাঠান" বাটন চাপা
    ↓
হোয়াটসঅ্যাপ খোলে, পুরো লিস্ট আগেই লেখা থাকে → Send চাপা
```

## অ্যাপের ফিচার তালিকা

### কোর ফিচার (v1 — এখন আছে)
- **ক্যাটাগরি ভিত্তিক আইটেম:** ১০টি ক্যাটাগরি, ~৯০টি আইটেম
- **"নিয়মিত বাজার" সেকশন:** সবচেয়ে সাধারণ আইটেমগুলো উপরে
- **এক-ট্যাপ অ্যাড:** ট্যাপ করলেই ডিফল্ট পরিমাণ সহ যোগ হয়
- **স্টেপার (+/−):** পরিমাণ বাড়ানো/কমানো, কিবোর্ড ছাড়াই
- **সার্চ:** বাংলায় নাম লিখলে ফিল্টার হয়
- **কাস্টম আইটেম:** তালিকায় নেই এমন কিছু লিখে যোগ করা যায়
- **স্টিকি ডক:** নিচে সবসময় দেখা যায় — কয়টা নির্বাচিত + পাঠানোর বাটন
- **রিভিউ শিট:** পাঠানোর আগে পুরো লিস্ট দেখা ও এডিট করা
- **WhatsApp ডিপ-লিংক:** `wa.me` দিয়ে সরাসরি নম্বরে খোলে, লিস্ট আগেই টাইপ করা
- **কপি বাটন:** অন্য জায়গায় পাঠানোর জন্য
- **বাংলা সংখ্যা:** সব সংখ্যা বাংলায় (১, ২, ৩...)
- **Mobile-first design:** মোবাইলে প্রধানত ব্যবহার হবে, সব ডিভাইসে responsive

### পরিকল্পিত ফিচার (v2 — পরে)
- গত লিস্ট মনে রাখা (localStorage)
- হোম স্ক্রিনে যোগ করার সুবিধা (PWA manifest)
- আইটেম ক্যাটালগ ব্যবহারকারী নিজে এডিট করতে পারবেন

## টেকনিক্যাল আর্কিটেকচার

```
index.html
├── <head>        → Google Fonts (Tiro Bangla + Hind Siliguri), CSS variables
├── <style>       → সম্পূর্ণ CSS (Claude design system: ক্রিম ব্যাকগ্রাউন্ড, ক্লে অ্যাকসেন্ট)
├── <body>        → Header, Search, Catalog sections, Custom input, Sticky dock, Sheet, Toast
└── <script>      → সম্পূর্ণ JS
    ├── CATALOG   → আইটেম ডেটা (JS array — এখানেই সব আইটেম/ক্যাটাগরি/একক/ডিফল্ট পরিমাণ)
    ├── State     → selected{}, customItems[]
    ├── Render    → renderCatalog(), renderSearch(), renderSheet()
    ├── Actions   → addItem(), changeQty(), sendWhatsApp(), copyList()
    └── UI        → dock, sheet, toast, search
```

### গুরুত্বপূর্ণ কনফিগ (JS-এর শুরুতে)
```javascript
const WHATSAPP_NUMBER = "8801628155191"; // +880 1628-155191
```

### CATALOG ডেটা স্ট্রাকচার
```javascript
{ cat:"ক্যাটাগরির নাম", eb:"ইংরেজি লেবেল", items:[
    ["আইটেমের নাম", "একক", ডিফল্ট_পরিমাণ, ধাপ, সর্বনিম্ন, নিয়মিত?],
    // নিয়মিত = 1 হলে "নিয়মিত বাজার" সেকশনেও দেখায়
]}
```
উদাহরণ:
```javascript
["আটা", "কেজি", 10, 5, 5, 1]   // নিয়মিত আইটেম, ডিফল্ট ১০ কেজি, ধাপ ৫
["করলা", "কেজি", 0.5, 0.5, 0.5] // অনিয়মিত, ডিফল্ট ০.৫ কেজি
```

## কোড মডিফাই করার নির্দেশনা (Claude Code-এর জন্য)

### আইটেম যোগ/বাদ দেওয়া
`CATALOG` array-এ সংশ্লিষ্ট ক্যাটাগরির `items` এ নতুন লাইন যোগ করো।
ফরম্যাট: `["নাম", "একক", ডিফল্ট, ধাপ, সর্বনিম্ন, (১ হলে নিয়মিত)]`

### নতুন ক্যাটাগরি যোগ করা
`CATALOG` array-এ নতুন object যোগ করো:
```javascript
{ cat:"নতুন ক্যাটাগরি", eb:"English Label", items:[ ... ] }
```

### WhatsApp নম্বর বদলানো
ফাইলের শুরুতে `WHATSAPP_NUMBER` ভেরিয়েবল।

### রঙ বদলানো
`:root` CSS variables-এ।

### কোনো কিছু না বোঝা গেলে
Omayer Hamdi-কে জিজ্ঞেস করো — তিনি বাংলায় উত্তর দেবেন।
